CSS

CSS (Cascading Style Sheets) - это язык стилей, который используется для оформления веб-страниц. CSS позволяет разработчикам задавать цвет, шрифт, размер и расположение элементов на странице.

Основные принципы CSS:

1. Селекторы: определяют, какие элементы на странице будут стилизованы.

2. Свойства: определяют, какие стили будут применены к выбранным элементам.

3. Значения: определяют конкретные значения свойств, такие как цвет, размер и т.д.

4. Каскадность: позволяет определять приоритетность стилей и их наследование от родительских элементов.

5. Наследование: позволяет наследовать стили от родительских элементов.

6. Единицы измерения: используются для определения размеров и расстояний на странице.
